Hong Kong Hospital Site Selection System Based on 2SFCA Algorithm

基于2SFCA Algorithm做了一个hospital site select system，可以根据需求调整以下四个维度的权重进行空间适宜度分析，最终得到选址适宜度得分，辅助决策。

1.Medical Urgency (Ai),2.Population Density 3.Vulnerable Groups 4.Transport Convenience

Core Methodology: Gaussian 2SFCA & MCE

Tech Stack: Python, Streamlit, Plotly, Scikit-learn
