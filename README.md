Hong Kong Hospital Site Selection System Based on 2SFCA Algorithm

#基于2SFCA Algorithm做了一个hospital site select system，可以根据需求调整以下四个维度的权重进行空间适宜度分析，最终得到选址适宜度得分，辅助决策.本项目成功解决了多源地理空间数据的 ID 匹配冲突，通过建立映射表实现了全香港200+ TPU 单元与路网阻抗矩阵的无缝集成.
1.Medical Urgency (Ai),2.Population Density 3.Vulnerable Groups 4.Transport Convenience
